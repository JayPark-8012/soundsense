# SoundSense QA 체크리스트

> 출시 전 전체 기능 검증용
> PRO 기능은 Debug 토글로 ON/OFF 전환하며 테스트

---

## 테스트 환경
- PRO 토글: 설정 > DEBUG > PRO 모드
- 테스트 계정: 게스트 (익명 — Firebase Anonymous Auth)
- 플랫폼: iOS / Android

## 1. 측정 화면
- [ ] 앱 진입 시 측정 화면 정상 표시

- [ ] 측정 중 dB 실시간 업데이트
- [ ] 측정 중 레벨 색상 변경 (5단계)
- [ ] 측정 중 FAB 색상 레벨 연동
- [ ] Fast/Slow 토글 전환
- [ ] 피크 홀드 바늘 표시
- [ ] 안전 노출 시간 표시
- [ ] Stop → 저장 바텀시트 자동 등장
- [ ] 저장 바텀시트 기본값 (위치 ON, 공유 ON)
- [ ] 저장하기 → 히스토리에 추가 확인
- [ ] 저장 안 함 → 바텀시트 닫힘
- [ ] 소음 가이드 이동
- [ ] 85dB 초과 → 햅틱 진동
