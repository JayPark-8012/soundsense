## 1. 앱 정체성

### 한 줄 정의
> "소음을 측정하고, 기록하고, 함께 도시의 소리를 그려가는 앱"

### 진화 전략 (가장 중요)
```
v1 — "가장 예쁜 소음 측정기"
     측정 + 히스토리로 유저를 모음
     지도는 내 측정 기록만 표시 (개인 소음 일지)
     목표: 스토어 평점 4.5+ / 다운로드 1만+

v2 — "내가 다녀온 곳의 소음 일지"
     커뮤니티 지도 오픈 (다른 유저 데이터 표시)
     장소 검색, 소음 레벨 필터, 클러스터링
     목표: MAU 리텐션 확보

v3 — "도시의 소음을 함께 기록하는 플랫폼"
     히트맵, 시간대별 분석, B2B 데이터
     목표: 네트워크 효과 발생
```

### 왜 이 순서인가
지도는 데이터가 없으면 가치가 0입니다.
측정 앱으로 유저를 먼저 모으고,
그 측정 데이터가 자연스럽게 지도를 채웁니다.
유저는 측정하러 오지만, 남는 이유는 지도 때문이 됩니다.

## 2. 핵심 결정 사항 (변경 불가)

### 로그인 없음
```
결정: v1부터 끝까지 로그인 없음
이유: 소음 측정기에 회원가입 → 이탈률 급등
대안: Firebase Anonymous Auth로 익명 기기 ID 발급
     유저는 로그인 없이 쓰고, 서버는 기기 단위 어뷰징 방지
```

### Firebase 구조 (확정)
```
결정: Anonymous Auth (로그인 없음)
구조:
  - 익명 기기 ID로 지도 어뷰징 방지
  - 재설치 시 히스토리 소실 허용
  - GDPR: 기기 ID 기준 데이터 삭제 가능
  - 위치 데이터를 처음부터 쌓아야 지도가 의미 있음
```

### 글로벌 출시 (영어 + 한국어 동시)
```
결정: 영어 우선, 한국어 동시 지원
이유: AdMob 단가 미국/캐나다/호주가 한국의 3~5배
     글로벌 소음 지도 앱 블루오션
```

### 측정 화면 광고 없음
```
결정: 측정 화면에는 광고 절대 금지
이유: 핵심 차별점 — 경쟁앱 전부 측정 화면에 광고 있음
     리뷰 점수 + 프리미엄 전환율 모두 영향
```

### Stop → 자동 저장 UX (확정)
```
결정: Stop 버튼 탭 시 자동으로 저장 바텀시트 등장
이유: 지도 플랫폼으로 진화하려면 저장률이 생명
스펙:
  - 별도 저장 버튼 없음
  - 기본값: 위치 ON, 지도 공유 ON, 메모 접힘
  - Start/Stop 버튼 항상 하단 고정
```

### 히스토리 잠금 방식 (확정)
```
결정: Last 7 Days 무료 (목록 + 상세 모두), 이후 목록부터 잠금
이유: 최근 데이터 무료 제공으로 앱 가치 체감 + PRO 전환 유도
스펙:
  - 무료: Last 7 Days (목록 + 상세 모두 접근)
  - 7일 이후: 목록에서 🔒 카드로 표시
  - PRO: All History 무제한 (목록 + 상세 모두 접근)
  - 버튼: [Last 7 Days] [All History 👑]
```

### 광고 전략 (확정)
```
결정: 배너 광고 영구 제거 — 전면 + 리워드만 사용
이유: 경쟁앱 1위 불만 = 광고가 UX 망침

전면 광고 (Interstitial):
  - 트리거: 세션 저장 완료 후, 3회마다 1회
  - 첫 저장에는 절대 표시 안 함
  - 최소 간격: 3분 (연속 저장 시 보호)

리워드 광고 (Rewarded):
  - 트리거: 세션 상세 "PDF 받기" 버튼
  - 유저 선택 — 강제 아님

PRO 유저:
  - 모든 광고 없음
```

### 지도 마커 (확정)
```
결정: 리플 애니메이션 마커 (일반 마커 금지)
이유: 소음 = 소리가 퍼져나가는 것 → 메타포 일치
     경쟁앱에서 본 적 없는 차별화 포인트
스펙:
  - 원의 크기: avgDb 비례 (40dB=최소, 100dB=최대)
  - 원의 색상: DbLevel 5단계
  - 원의 속도: dB 클수록 빠름
  - 투명도: 퍼질수록 0으로 소멸
  - 클러스터링: v2 예정
```

### 기기 캘리브레이션 (확정)
```
결정: 설정 화면에 슬라이더 (-10dB ~ +10dB)
제공: 무료 (신뢰도 확보용)
UI: 심플하게 슬라이더 하나만
이유: 초기 리뷰 "측정값 이상해요" 방어
     소음 지도 데이터 신뢰도 보장
```

### Noise Card (확정)
```
결정: 측정 결과를 예쁜 카드 이미지로 생성
카드 구성:
  - 브랜드 + 레벨 색상 + 평균dB
  - 레벨 텍스트 + 날짜/시간/위치 + 면책문구
무료: 워터마크 있음
PRO: 워터마크 없음
카메라 권한 불필요 (이미지 생성 방식)
이유: 층간소음 증거용, SNS 바이럴, 브랜딩
```

## 4. 화면별 UX 기획

---

### 화면 1: 메인 측정 화면

#### 레이아웃 원칙
```
Column
├── 상단: 앱바 (배경 = AppColors.background, 끊김 없음)
├── 중단: 레벨 배지 + 게이지 + dB 숫자 + MIN/AVG/MAX
│         (Expanded — 남은 공간 모두 사용)
└── 하단: Start/Stop 버튼 (항상 고정, 스크롤 영향 없음)
```

#### Stop → 저장 UX 플로우 (중요)
```
[측정 중]
  Stop 버튼 탭
      ↓
  자동으로 저장 바텀시트 등장
  (별도 저장 버튼 없음 — 저장을 쉽게!)
      ↓
  바텀시트 기본값:
  - 위치: 현재 위치 ON (자동)
  - 지도 공유: ON
  - 메모: 접힌 상태 (선택사항 강조)
      ↓
  [💾 저장하기] 메인 CTA
  [저장 안 함] 작게 표시
```

#### 왜 이 UX인가
```
기존 문제:
  - 상단 우측 작은 아이콘 = 저장 기능 존재 자체를 모름
  - 저장을 일부러 어렵게 만든 느낌
  - 지도 플랫폼으로 진화하려면 저장률이 생명

해결:
  - Stop = 저장으로 자연스럽게 연결
  - 기본값 ON으로 저장 마찰 최소화
  - 메모는 선택사항으로 시각적 표현
```

#### 소음 레벨 구간
```
0  ~ 40dB │ 🔵 파랑 │ Very Quiet  │ 매우 조용해요
40 ~ 55dB │ 🟢 초록 │ Quiet       │ 조용한 환경
55 ~ 70dB │ 🟡 노랑 │ Moderate    │ 보통 소음
70 ~ 85dB │ 🟠 주황 │ Loud        │ 시끄러운 환경
85~100dB  │ 🔴 빨강 │ Very Loud   │ 청력 주의
100dB+    │ 🔴 빨강 │ Dangerous!  │ 즉시 피하세요
```

## 5. 디자인 원칙

### 브랜드 방향성
**"정밀 계측 장비의 신뢰감 + 프리미엄 앱의 세련됨"**

경쟁앱들이 못생겼다는 게 우리의 기회.
측정값을 누구나 이해할 수 있도록 단순화하되,
손에 들고 싶은 앱을 만든다.

### UI 퀄리티 기준
```
✅ 모든 전환 애니메이션 (200~300ms)
✅ dB 숫자 변화 부드럽게 (급격한 점프 없음)
✅ 햅틱 피드백 (버튼, 경고)
✅ 다크 테마 기본
✅ 레벨 색상 변화 크로스페이드
✅ 게이지 바늘 스프링 애니메이션
✅ 측정 화면 광고 없음
```

### 컬러 팔레트
```
background      #0D1117   딥 블랙 (메인 배경)
surface         #161B22   카드/시트
surfaceVariant  #21262D   입력창, 구분선
accent          #58A6FF   블루 (활성, CTA)

levelSilent     #388BFD   파랑  (0~40dB)
levelQuiet      #3FB950   초록  (40~55dB)
levelModerate   #D29922   노랑  (55~70dB)
levelLoud       #DB6D28   주황  (70~85dB)
levelDanger     #F85149   빨강  (85dB+)

proGold         #D4A017   골드 (PRO 배지)
```

## 7. 무료 vs 프리미엄 (최종 확정)

### 무료 (Free)
```
실시간 측정            무제한
세션 저장              무제한
히스토리              Last 7 Days (목록 + 상세 모두 접근)
히스토리 카드          7일 이후 🔒 카드로 표시
소음 지도              내 기록 보기
기기 캘리브레이션      무료 제공
Noise Card 공유        워터마크 있음
광고                   전면(3회 저장마다) + 리워드(선택)
측정 화면 광고         없음 ← 차별점
```

### 프리미엄 PRO ($2.99/월 · $19.99/년 · 7일 무료 체험)
```
한국: ₩3,900/월 · ₩24,900/년

All History 무제한 (목록 + 상세 모두 접근)
세션 라인 차트
월간 PDF 리포트 무제한
CSV 내보내기
Noise Card 워터마크 없음
모든 광고 제거
주간 소음 알림
홈화면 위젯
```

### PRO 유도 원칙
```
✅ 잠긴 기능 흐리게 보여주고 탭 시 설명
✅ "PRO에서 더 볼 수 있어요" 톤
✅ 7일 무료 체험 전면 강조
❌ 측정 화면 팝업 절대 금지
❌ 세션당 2회 이상 팝업 금지
❌ 강압적 UI 금지
```

EN:

```
✅ 위치 데이터 익명 처리

## 9. 스토어 출시 전략

### ASO 키워드
```
영어: noise meter, decibel meter, sound level meter,
      db meter, noise map, hearing protection
한국어: 소음측정, 데시벨, 층간소음, 소음계,
        dB측정기, 소음지도, 청력보호
```

### 스크린샷 구성 (5장)
```

## 12. 온보딩 스펙

### 공통
```
스타일:     풀스크린 슬라이드 (PageView)
건너뛰기:   전체 건너뛰기 가능 (우상단 Skip 버튼)
권한 요청:  온보딩 중에 (Step 3 마이크, Step 4 위치)
재확인:     kDebugMode일 때 설정 화면에
            [온보딩 다시 보기] 버튼 표시
```

### Step 1 — 앱 소개
```
배경: AppColors.background
중앙 큰 🎙️ 아이콘 (accent 색상 글로우)
타이틀: "Measure the world around you"
서브:   "Real-time noise measurement,
         history & noise map"
하단: [Get Started] 버튼 + Skip
```

### Step 2 — 소음 기준 안내
```
타이틀: "Know your noise levels"
레벨별 색상 카드 5개:
  🔵 Very Quiet   ~40dB  도서관, 새벽
  🟢 Quiet        ~55dB  조용한 사무실
  🟡 Moderate     ~65dB  일반 대화, 카페
  🟠 Loud         ~75dB  번화가, 음식점
  🔴 Dangerous    85dB+  청력 주의 (강조)
하단 WHO 기준 한 줄:
  "WHO: 85dB+ for 8hrs may cause hearing damage"
[Next] 버튼 + Skip
```

### Step 3 — 마이크 권한
```
중앙 큰 🎙️ 아이콘
타이틀: "Let's start measuring"
서브:
  "✓ No audio is recorded"
  "✓ Data stays on your device"
[Allow Microphone] → 시스템 권한 팝업
거부 시: 측정 화면에서 다시 안내 (에러 아님)
[Skip] 작게 하단
```

### Step 4 — 위치 권한
```
중앙 큰 🗺️ 아이콘
타이틀: "Tag your measurements"
서브:
  "Auto-tag locations & contribute
   to the noise map"
  "You can skip this anytime"
[Allow Location] → 시스템 권한 팝업
[Skip] 동등하게 표시 (부담 없이)
완료 → 메인 측정 화면으로 이동
```
